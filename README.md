 # 후르츠 박스 (Apple Game) - AWS 실습

1. **E2c생성**: AWS EC2 인스턴스를 생성하고 Node.js 환경을 설정하여 백엔드 서버를 가동했습니다.
2. **백엔드 개발**: JavaScript(Express)를 사용하여 점수 저장 및 랭킹 조회를 위한 시스템를 구현했습니다.
3. **데이터베이스**: AWS RDS(MySQL)와 서버를 연동하여 실시간 랭킹 시스템을 구축했습니다.
4. **프론트엔드**: 효율적인 관리를 위해 하나의 HTML 파일에 게임 로직과 UI를 담아 개발했습니다.
5. **CI/CD 자동화**: GitHub Actions를 설정하여 코드 커밋 시 S3 버킷으로 자동 업로드되도록 구현했습니다.
6. **S3 호스팅**: S3 정적 웹 사이트 호스팅을 통해 index.html 파일이 즉시 브라우저에서 실행되게 했습니다.
7. **이미지화(AMI)**: 설정이 완료된 서버를 AMI로 이미지화하여, 동일한 환경의 서버를 즉시 복제하고 확장할 수 있는 기반을 마련했습니다.
8. **오토 스케일링**: 시작 템플릿과 ASG를 설정하여 트래픽 변화에 유연하게 대응하고, ALB(로드 밸런서)를 통해 부하를 분산하는 고가용성 구조를 완성했습니다.
9. **네트워크**: 로드 밸런서(ALB) DNS를 통해 S3와 EC2 사이의 안정적인 데이터 통신을 연결했습니다.
